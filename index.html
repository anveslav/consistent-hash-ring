URL Routing to Message Queues

🐰 How RabbitMQ Consistent Hash Exchange Works:

  • Routing Keys: Each URL/domain is used as a routing key for message distribution
  • Queues: Messages are distributed across multiple queues based on hash of routing key
  • Weight: Queue capacity - higher weight = processes more messages
  • Hash Ring: hash(routing_key) % total_buckets → determines which queue gets the message
  • Consistency: Same routing key always routes to the same queue (useful for ordering)
